# Site Web : Révision Programmation Orientée Objet et Java
Un site pédagogique dédié au révision du cours de POO - Java de L2 informatique CY PARIS UNIVERSITE
Ce projet est un mini-site web statique conçu comme un outil de révision 
pour la Programmation Orientée Objet (Java) et la modélisation UML.

Il a été réalisé en pur HTML5 (sans CSS, conformément à la consigne) 
et respecte les standards sémantiques du W3C.

Le contenu technique s'appuie sur :
- Le polycopié de cours "Aide mémoire UML & Java" de M. Marc Lemaire.
- Les sujets de TD de L2 Informatique (TD 5, 6 et 8).
